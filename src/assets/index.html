Add a provider

Any CLI becomes a provider with a TOML manifest — which binary to run and how to read a balance out of its output. Saved to ~/.config/utiman/providers/. See the manifest reference.









  
utiman never sees your passwords — each CLI keeps its own credentials in the
  OS keychain, and this dashboard only answers requests from this machine.
